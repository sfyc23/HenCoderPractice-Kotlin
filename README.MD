# HenCoder 练习项目- Kotlin 版

---

## 一个用 Kotlin 实现的 HenCoder 练习项目集合(已完成到 2.1)
页面如下图所示:

![main][1] ![1-1][2] ![1-2][3]

![1-3][4] ![1-4][5] ![1-5][6]


1. 在完成的基础上，多加上了多属性相关变化，或者注释。以更方便，直观的了解相关的属性；
2. 使用 Anko Layouts 代替 XML 做布局。大大减小 xml 文件的数量；
3. 将数据放在一起，一家人在一起，就是要整整齐齐。

[体验 demo ](https://fir.im/ven7)
